This file contains classes and methods for policies and courses extractor

Catalog Engine for Gettysburg College (catalog\+\_\+engine) is the engine which extracts content from Gettysburg website, together with Gettysburg website policies and courses api, and generate the appropriate courses content catalog for an academic year. More documentations can be viewed at {\ttfamily ../../doc/} folder.
